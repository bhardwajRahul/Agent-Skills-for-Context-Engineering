Specification engineering for autonomous work.

Persistence is not correctness. As RL and agent harnesses extend task horizons, ambiguous completion criteria give an optimizing agent more opportunities to return an answer-shaped near miss.

This skill turns a launch prompt into a concrete specification: exact success, named non-solutions, evidence requirements, adversarial verification, and a return gate. Select a case below to compare and copy the complete prompts.
